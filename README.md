# Touch ON/OFF Sensor Switch using 555 Timer

## Overview
This project implements a touch-activated ON/OFF switch using an IC 555 timer configured in bistable mode. The circuit eliminates the need for mechanical switches by using touch as an input trigger.

## Working Principle
The IC 555 timer operates in bistable mode, where the output remains in one of two stable states (HIGH or LOW).  
- A touch input acts as a trigger signal to toggle the state of the output.  
- When triggered, the output switches between ON and OFF states.  
- An LED is used as the load to visually indicate the output state.  

The touch mechanism works on the principle of capacitive sensing, where the human body introduces a small capacitance that is sufficient to trigger the circuit.

## Components Used
